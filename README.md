# Networking Labs

Networking fundamentals documentation and lab notes. Covers subnetting, protocols, traffic analysis, and firewall concepts with a focus on security implications.

## Topics

| Topic | Description | Status |
|---|---|---|
| Subnetting | CIDR, subnet masks, worked examples | Planned |
| TCP/IP & UDP | How packets work, three-way handshake, flags | Planned |
| DNS | Resolution process, record types, DNS security | Planned |
| HTTP/HTTPS & TLS | Request/response cycle, TLS handshake, certificates | Planned |
| ARP & DHCP | Layer 2 fundamentals, ARP spoofing awareness | Planned |
| Wireshark Labs | Capture filters, display filters, traffic analysis | Planned |
| Firewall Rules | ufw, iptables, default-deny principles | Planned |

## Tools Used

- Wireshark
- Nmap
- tcpdump
- netstat / ss

## Environment

Labs performed on isolated virtual networks using VirtualBox/VMware. No external systems involved.
